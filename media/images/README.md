# Media Images Directory

This directory contains all images and media files for the induction annealing machine project.

## Image Files

The following images should be present in this directory:

### Hardware Images
- `general_view.jpg` - Overall view of the complete machine
- `zvs_driver.jpg` - ZVS flyback driver board (close-up)
- `T10_V1.4.jpg` - Controller board (TTGO T1.4 V1.4) - *Already present*
- `controller_front.jpg` - Front view of the controller with display
- `controller_display.jpg` - Controller showing the display interface in operation
- `rear_view.jpg` - Rear view of the machine showing cooling setup
- `cooling_setup.jpg` - Detailed view of the cooling system

### Results Images
- `annealing_result.jpg` - Example showing an annealed cartridge case
- `case_comparison.jpg` - Before/after comparison of annealed cases

## Download Instructions

If images are missing, you can download them using:

1. **Automatic method**: Run `../download_images.sh` from the project root
2. **Manual method**: 
   - Visit the original Telegraph article
   - Right-click and save each image with the appropriate filename

## Notes

- All images are stored locally to ensure the documentation is self-contained
- Images should be in JPG format for compatibility
- Maintain reasonable file sizes for web viewing (recommended: < 500KB per image)
